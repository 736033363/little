图片优化

1. 图片色彩过于丰富且无透明要求，建议使用jps并保存为高质量 2. 图片色彩过于丰富且有透明、半透明、阴影时，建议采用png24格式 3. 色彩不太丰富时无论有无透明要求，采用png8格式，大多数情况采用这样格式 4. 图片有动画时，只能使用gif格式 5. 可以使用工具对图片再次压缩，但前提不能影响色彩和透明
