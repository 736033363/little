其他

  • 修改最后一次注释
  • Git提交记住用户名和密码（需验证） 执行push前执行 git config --global credential.helper store（省略--global则只在该项目内生效），下次就不会要求输入用户名和密码。可参考：https://blog.csdn.net/youanyyou/article/details/78992990。如果每个项目的用户名密码不同，可能需要多试几次，执行该命令记录即可。
